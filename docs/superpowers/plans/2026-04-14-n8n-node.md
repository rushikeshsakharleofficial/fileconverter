# n8n Community Node — PixConvert Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Build and publish `n8n-nodes-pixconvert`, a community node that exposes all 35 PixConvert API endpoints in n8n with credential-based base URL config, binary + URL output modes, and 5 workflow templates.

**Architecture:** Standalone TypeScript repository compiled to `dist/`. Single `PixConvert` node with Resource → Operation → Parameters UI pattern. Shared `transport.ts` handles all HTTP, FormData building, and binary/URL output logic. Operations split into 8 category files exporting `INodeProperties[]` arrays.

**Tech Stack:** TypeScript 5, n8n-workflow (peer dep), form-data, got, Jest + ts-jest, Gulp (icon copy), eslint

```typescript
import FormData from 'form-data';
import https from 'https';
import http from 'http';
import { URL } from 'url';

export interface PixConvertRequestOptions {
  baseUrl: string;
  endpoint: string;
  formFields?: Record<string, string | undefined>;
  fileBuffer?: Buffer;
  fileName?: string;
  fileMimeType?: string;
  fileBuffer2?: Buffer;
  fileName2?: string;
  outputMode: 'binary' | 'url';
}

export interface PixConvertBinaryResult {
  type: 'binary';
  buffer: Buffer;
  mimeType: string;
  fileName: string;
}

export interface PixConvertUrlResult {
  type: 'url';
  url: string;
}

export type PixConvertResult = PixConvertBinaryResult | PixConvertUrlResult;

/** Build a FormData object from a string field map, skipping empty/undefined values */
export function buildFormData(
  fields: Record<string, string | undefined>,
): FormData {
  const form = new FormData();
  for (const [key, value] of Object.entries(fields)) {
    if (value !== undefined && value !== '') {
      form.append(key, value);
    }
  }
  return form;
}

/** Parse an error message out of a PixConvert API error response body */
export function parsePixConvertError(body: string): string {
  if (!body) return 'Unknown error from PixConvert API';
  try {
    const parsed = JSON.parse(body);
    return parsed.error ?? 'Unknown error from PixConvert API';
  } catch {
    return body;
  }
}

/** Execute a PixConvert API request */
export async function pixConvertRequest(
  options: PixConvertRequestOptions,
): Promise<PixConvertResult> {
  const { baseUrl, endpoint, formFields = {}, fileBuffer, fileName, fileMimeType, fileBuffer2, fileName2, outputMode } = options;

  // Check file size before sending (50MB limit)
  const MAX_BYTES = 50 * 1024 * 1024;
  if (fileBuffer && fileBuffer.length > MAX_BYTES) {
    throw new Error(`File exceeds the 50MB PixConvert API limit (${(fileBuffer.length / 1024 / 1024).toFixed(1)}MB)`);
  }
  if (fileBuffer2 && fileBuffer2.length > MAX_BYTES) {
    throw new Error(`Second file exceeds the 50MB PixConvert API limit`);
  }

  const form = buildFormData(formFields);

  if (fileBuffer && fileName) {
    form.append('file', fileBuffer, { filename: fileName, contentType: fileMimeType ?? 'application/octet-stream' });
  }
  if (fileBuffer2 && fileName2) {
    form.append('file2', fileBuffer2, { filename: fileName2, contentType: 'application/octet-stream' });
  }

  const urlStr = `${baseUrl.replace(/\/$/, '')}/api/v1/${endpoint}${outputMode === 'url' ? '?output=url' : ''}`;
  const parsed = new URL(urlStr);
  const isHttps = parsed.protocol === 'https:';
  const lib = isHttps ? https : http;

  return new Promise((resolve, reject) => {
    const req = lib.request(
      urlStr,
      {
        method: 'POST',
        headers: form.getHeaders(),
      },
      (res) => {
        const chunks: Buffer[] = [];
        res.on('data', (chunk: Buffer) => chunks.push(chunk));
        res.on('end', () => {
          const body = Buffer.concat(chunks);
          const statusCode = res.statusCode ?? 0;

          if (statusCode === 429) {
            reject(new Error('Rate limit reached (10 req/sec). Retry after 1 second.'));
            return;
          }

          if (statusCode < 200 || statusCode >= 300) {
            reject(new Error(`PixConvert API error: ${parsePixConvertError(body.toString())}`));
            return;
          }

          if (outputMode === 'url') {
            try {
              const json = JSON.parse(body.toString());
              resolve({ type: 'url', url: json.url });
            } catch {
              reject(new Error('Expected JSON url response but got invalid body'));
            }
            return;
          }

          const contentType = res.headers['content-type'] ?? 'application/octet-stream';
          const disposition = res.headers['content-disposition'] ?? '';
          const match = disposition.match(/filename="?([^";\n]+)"?/);
          const outFileName = match ? match[1] : `output.${contentType.split('/')[1] ?? 'bin'}`;

          resolve({ type: 'binary', buffer: body, mimeType: contentType, fileName: outFileName });
        });
      },
    );

    req.on('error', reject);
    form.pipe(req);
  });
}
```

  async execute(this: IExecuteFunctions): Promise<INodeExecutionData[][]> {
    const items = this.getInputData();
    const returnData: INodeExecutionData[] = [];
    const credentials = await this.getCredentials('pixConvertApi');
    const baseUrl = credentials.baseUrl as string;

    for (let i = 0; i < items.length; i++) {
      const resource = this.getNodeParameter('resource', i) as string;
      const operation = this.getNodeParameter('operation', i) as string;
      const inputSource = this.getNodeParameter('inputSource', i) as 'binary' | 'url';
      const outputMode = this.getNodeParameter('outputMode', i) as 'binary' | 'url';

      const endpoint = ALL_ENDPOINTS[operation];
      if (!endpoint) {
        throw new NodeOperationError(this.getNode(), `Unknown operation: ${operation}`);
      }

      const getParam = (name: string) => this.getNodeParameter(name, i, '');
      const formFields = getExtraFields(getParam, resource, operation);

      let fileBuffer: Buffer | undefined;
      let fileName: string | undefined;
      let fileMimeType: string | undefined;
      let fileBuffer2: Buffer | undefined;
      let fileName2: string | undefined;

      if (inputSource === 'binary') {
        const propName = this.getNodeParameter('binaryPropertyName', i, 'data') as string;
        const binaryData = items[i].binary?.[propName];
        if (!binaryData) {
          throw new NodeOperationError(this.getNode(), `No binary data found in property "${propName}"`, { itemIndex: i });
        }
        fileBuffer = await this.helpers.getBinaryDataBuffer(i, propName);
        fileName = binaryData.fileName ?? 'input.bin';
        fileMimeType = binaryData.mimeType;
      } else {
        const url = this.getNodeParameter('fileUrl', i, '') as string;
        formFields['url'] = url;
      }

      // Second file for compare/sign operations
      if (resource === 'security' && (operation === 'comparePdf' || operation === 'signPdf') && inputSource === 'binary') {
        const prop2 = this.getNodeParameter('binaryPropertyName2', i, 'data2') as string;
        const bin2 = items[i].binary?.[prop2];
        if (bin2) {
          fileBuffer2 = await this.helpers.getBinaryDataBuffer(i, prop2);
          fileName2 = bin2.fileName ?? 'input2.bin';
        }
      }

      try {
        // Retry once on 429
        let result;
        try {
          result = await pixConvertRequest({ baseUrl, endpoint, formFields, fileBuffer, fileName, fileMimeType, fileBuffer2, fileName2, outputMode });
        } catch (err: unknown) {
          const msg = err instanceof Error ? err.message : String(err);
          if (msg.includes('Rate limit')) {
            await new Promise((r) => setTimeout(r, 1000));
            result = await pixConvertRequest({ baseUrl, endpoint, formFields, fileBuffer, fileName, fileMimeType, fileBuffer2, fileName2, outputMode });
          } else {
            throw err;
          }
        }

        if (result.type === 'url') {
          returnData.push({ json: { url: result.url } });
        } else {
          const binaryData = await this.helpers.prepareBinaryData(result.buffer, result.fileName, result.mimeType);
          returnData.push({ json: {}, binary: { data: binaryData } });
        }
      } catch (err: unknown) {
        const msg = err instanceof Error ? err.message : String(err);
        throw new NodeOperationError(this.getNode(), msg, { itemIndex: i });
      }
    }

    return [returnData];
  }
}

# n8n-nodes-pixconvert

An n8n community node for [PixConvert](https://github.com/rushikeshsakharleofficial/fileconverter) — a self-hosted PDF and image processing API.

## Features

- **35 operations** across 8 categories: Organize PDF, Optimize PDF, Convert to/from PDF, Edit PDF, PDF Security, Image Conversion, Media
- **Binary or URL output** — pipe processed files directly to S3, email, Google Drive, or return a download URL
- **No authentication required** — configure once with your server's base URL

## Installation

In your n8n instance, go to **Settings → Community Nodes** and install:

```
n8n-nodes-pixconvert
```

Or install via npm in your n8n root:

```bash
npm install n8n-nodes-pixconvert
```

## Setup

1. Add a **PixConvert API** credential
2. Set **Base URL** to your server (e.g. `https://your-server.com`)
3. The node tests the connection with `GET /api/v1/health`

## Deploying the API

The PixConvert API requires a Linux server with:
- Node.js 18+
- Ghostscript (`gs`)
- LibreOffice headless (`libreoffice`)
- Tesseract OCR (`tesseract`)

See the [PixConvert server repo](https://github.com/rushikeshsakharleofficial/fileconverter) for deployment instructions.
